Документ "Переоценка основных средств"

Документ "Переоценка основных средств" служит для отражения в бухгалтерском учету операции по ежегодной обязательной переоценке основных средств числящихся на балансе предприятия на счете: "Счета учета основных средств" ("01").

Документ формирует проводки по изменению первоначальной стоимости и накопленного износа в корреспонденции со счетом "Корректировка по переоценке имущества" ("85.10").

Программа самостоятельно контролирует существования единственного проведенного документа "Переоценка основных средств" в течении одного года. При попытке проведения документа "Переоценка основных средств" при существущем проведенном документе программа выводит соответствующее предупреждение и запрещает проведение документа. Дата и момент времени проведения документа назначается программой автоматически.

Документ обслуживает два режима ввода данных:

  • Автоматическая загрузка коэффициентов и периодов для переоценки;
  • Ручной ввод коэффициентов и периодов для переоценки;

Автоматическая загрузка коэффициентов и периодов для переоценки

Для автоматической загрузки коэффициентов и периодов переоценки необходимо в реквизите "Имя файла коэффициентов переоценки" указать путь к "mxl" файлу с даннымы. Скопровать этот файл можно с диска "ИТС". После указания пути к файлу необъодимо нажать кнопку "Загрузить внешние данные" на панели основных действий формы документа. Загруженные данные автоматически помещаются в табличные части на закладках "Периоды" и "Коэффициенты".

Ручной ввод коэффициентов и периодов для переоценки

В табличной части на закладке "Периоды" необходимо ввести 8 строк указывая по каждой строке начало и конец периода переоценки. В первой строке, в колонке "Начало периода" дату указывать не нужно.

В табличной части на закладке "Коэффициенты" необходимо ввести строки с указанием счета учета основных средств и коэффициента в графы с "Коэффициент 1" по "Коэффициент 8". Табличная часть "Коэффициенты" не должна содержать двух строк с одинаковым счетом учета основных средств.

После того, как необходимые данные по периодам и коэффициентам переоценки внесены в документ, расчет сумм переоценки выполняется автоматически при нажатии кнопки "Выполнить переоценку".

Табличная часть "Переоценка" может быть заполнена вручную без заполнения  таблиц на закладках "Периоды" и "Коэффициенты".
