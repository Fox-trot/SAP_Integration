Документ «Сертификация номенклатуры»

Документ «Сертификация» служит для отражения факта сертификации материалов, товаров, продукции, либо отказа в сертификации.

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа с помощью кнопки «Операция».

В документе заполняется информация о номенклатуре, серии номенклатуры. Обязательно заполняется поле «Результат сертификации». Поле может принимать два значения - «Сертификация» или «Отказано в сертификации». Если выбрано значение «Отказано в сертификации», то дополнительные поля документа недоступны.

Для того чтобы отразить факт сертификации, следует выбрать значение поля «Результат сертификации» как «Сертификация».

- поле «Нормативный документ». Поле заполняется системой автоматически, на основании данных регистра сведений «Нормативные документы сертификации номенклатуры», но может быть выбрано вручную, из справочника «Нормативные документы сертификации номенклатуры».

- табличная часть, содержащая следующие реквизиты:

- Показатель анализа - показатель, по которому производилась оценка пробы в ходе анализов.

- Значение показателя - фактическое значение данного показателя по результатам измерений.

- Флаг соответствия указанного значения нормативу - рассчитывается автоматически при выборе значения показателя по данным о нормативном значении, указанном в соответствующем показателе. Допускается изменять данное значение вручную. Это может использоваться, в случае, если значение какого-то показателя нельзя описать стандартными средствами для автоматического определения его соответствия нормативу.

- информация о периоде проведения испытаний.

Документ «Сертификация номенклатуры» может быть введен на основании документов «Поступление товаров и услуг», «Заявка на сертификацию», «Акт отбора проб номенклатуры». Ввод на основании «Поступления товаров и услуг» производится посредством выбора из автоматически сформированного списка пары «серия-номенклатура». В список попадают только те серии, для которых необходимо провести внешнюю сертификацию, но сертификация не оформлена, а также не оформлена «Заявка на сертификацию». Ввод на основании документов «Заявка на сертификацию номенклатуры», «Акт отбора проб номенклатуры» осуществляется путем переноса значений всех соответствующих полей.
