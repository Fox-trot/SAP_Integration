Выбор организаций

Форма вызывается при нажатии кнопки выбора в поле ввода «Организации» формы «Регламентированная и финансовая отчетность» и стартовой формы бухгалтерских отчетов.

На командной панели, расположенной над списком организаций, размещен набор кнопок и подменю, предназначенный для облегчения работы со списком.

Для изменения текущего элемента списка воспользуйтесь кнопкой  «Изменить».

Для установки отбора в списке воспользуйтесь группой кнопок управления отбором.

Для перехода к информации в связанных со справочником объектах воспользуйтесь подменю «Перейти».

Для изменения пометок одновременно у всех организаций используйте кнопки «Снять флажки» и «Установить флажки».

Подбор организаций для составления бухгалтерской отчетности

Для составления бухгалтерской отчетности по нескольким организациям необходимо в списке организаций отметить нужные организации.

Кнопка «Обособленные предприятия» появляется в командной панели в том случае, если форма выбора организаций вызвана из стартовой формы бухгалтерского отчета. При нажатой кнопке в списке организаций будут оставлены только те организации, у которых в реквизите «Головная организация» в форме элемента справочника «Организации» указана организация, выбранная в стартовой форме.

При нажатии кнопки «OK» список выбранных организаций будет перенесен в бухгалтерский отчет и использован при заполнении отчета.
