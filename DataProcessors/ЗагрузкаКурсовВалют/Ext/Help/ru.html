Обработка "Загрузка курсов валют"

С помощью данной обработки можно произвести загрузку актуальных курсов валют. Загрузка курсов может производиться в платном и бесплатном режимах.
В бесплатном режиме доступны:
- загрузка курса из Интернет за ближайшее (не превышающее текущую дату) число
- загрузка курса с диска ИТС за период с начала предыдущего года до конца предыдущего месяца

В платном режиме доступна загрузка из Интернет за произвольную дату, за произвольный период.

Обработка вызывается из справочника "Валюты" при нажатии на кнопку "Загрузить курсы".

При нажатии кнопки "Настройка" будут отображены поля для ввода кода доступа к платным услугам,  для указание буквы привода дисков CD-ROM в который предполагается установка диска ИТС, а также опция "Загружать ежедневно".
При выборе последней опции, ежедневно, при запуске системы будет происходить загрузка курсов валют.
Для корректной работы данного сервиса необходимо подобрать те валюты из справочника "Валюты", по которым необходимо загружать курсы, при наличии кода доступа к платным сервисам - ввести его, после чего нажать кнопку "Сохранить настройки".

С помощью кнопки "Подбор" можно подобрать те валюты из справочника "Валюты", по которым необходимо загружать курсы.

С помощью кнопки "Заполнить" можно заполнить список валют всеми валютами, которые зарегистрированы в справочнике "Валюты".

Загрузка курсов производится при нажатии на кнопку "Загрузить". После успешной загрузки информация о курсах валют автоматически записывается в регистр сведения для каждой валюты.

Кнопка "Очистить" очищает список валют и их курсов.
